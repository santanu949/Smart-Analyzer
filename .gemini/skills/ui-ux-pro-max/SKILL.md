# UI/UX Pro Max Skill

AI-powered design intelligence with 67 UI styles, 161 color palettes, 57 font pairings, 99 UX guidelines, and 25 chart types across 15+ tech stacks.

## How to Use This Skill

Follow this workflow to improve the UI/UX of the project:

### Step 1: Generate Design System
Run the following command to get comprehensive design recommendations for the project:

```bash
python ui-ux-pro-max-skill/src/ui-ux-pro-max/scripts/search.py "Smart Contract Security Analyzer AI platform dark mode premium high-fidelity" --design-system -p "NexusScan AI"
```

### Step 2: Supplement with Detailed Searches
Use domain searches to get additional details like styles, color palettes, and UX best practices:

```bash
python ui-ux-pro-max-skill/src/ui-ux-pro-max/scripts/search.py "<keyword>" --domain <domain>
```

Available domains: `product`, `style`, `typography`, `color`, `landing`, `chart`, `ux`, `react`, `web`, `prompt`.

### Step 3: Implement & Review
Synthesize the design system and detailed searches to implement the design. Use the UX checklist in the skill documentation to validate the implementation.
